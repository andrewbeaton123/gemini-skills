# Python Code Reviewer

A rigorous code reviewer for Python projects, designed to catch bugs, enforce style, optimize performance, and improve maintainability.

## Framework Portability

This skill is a **Persona injection**. To port it to other frameworks:
1. Copy the instructions from `SKILL.md`.
2. Apply them as a `System Prompt` or `Assistant Persona`.
3. Optionally, use it as a "Reviewer" in a multi-agent workflow (e.g., Code → Reviewer → Refactor loop).
